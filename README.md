# checked_pattern_img_bookmark

## 사용법

#### git clone 사용시
```
git clone https://github.com/Sihan-Son/checked_pattern_img_bookmark.git
cd checked_pattern_img_bookmark
python bookmark.py
```

#### Download Zip 사용시

checked_pattern_img_bookmark 폴더로 이동  
`cmd` 등에서 

```
python bookmark.py
```
